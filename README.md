# Markdown Lite iOS

iPad版 Markdown Lite - Google Driveと連携するMarkdownエディター

## 概要

Mac版 [markdown-lite](https://github.com/TMrev-z/markdown-lite) のiPad対応版です。

## 主な機能

- リアルタイムMarkdownプレビュー
- Google Drive連携（閲覧・編集・新規作成）
- 自動保存（1秒デバウンス）
- フォルダ/ファイル管理
- 検索機能（ファイル名・内容）
- 画像挿入・リサイズ

## 技術スタック

- SwiftUI
- Google Drive API
- iOS/iPadOS 15.0+

## 開発環境

- Xcode 15.0+
- macOS 14.0+
- Apple Developer Account
